# ☁️ Weather App - Real-Time Forecasts

A modern, high-performance Flutter application that provides real-time weather updates, accurate forecasts, and location-based weather insights with a sleek and minimalist UI.

## 🚀 Features

* **Live Weather Data:** Instant access to temperature, humidity, and wind speed.
* **Location Intelligence:** Automatically fetches weather updates based on your current geographical location.
* **Extended Forecasts:** Daily and upcoming weather predictions to help you plan ahead.
* **Minimalist UI:** A clean, professional, and easy-to-read dashboard for financial-level clarity.
* **Search Functionality:** Quickly find weather conditions for any city globally.

---

## 🛠️ Tech Stack & Tools

* **Architecture:** **MVVM (Model-View-ViewModel)** for a clear separation of concerns.
* **State Management:** **Riverpod** for a robust, testable, and reactive state handling.
* **Navigation:** **AutoRoute** for strongly-typed and easy-to-manage routing.
* **Networking:** **Dio** for handling API requests with high efficiency.
* **API Integration:** Connects with professional weather service APIs for high-accuracy data.
