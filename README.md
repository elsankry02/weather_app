# ☁️ Weather App - Real-Time Forecasts

# 🚀 Features
* **Live Weather Data:** Fetches temperature, humidity, and wind speed.
* **Location Services:** Get weather updates based on your current location.
* **Daily Forecasts:** Stay updated with upcoming weather conditions.
* **Minimalist UI:** Clean and easy-to-read weather dashboards.

# 🛠️ Tech Stack
* **Architecture:** MVVM.
* **State Management:** Riverpod.
* **Navigation:** AutoRoute.
* **API:** Integration with high-accuracy weather APIs.
